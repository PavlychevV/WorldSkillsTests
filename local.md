# Конфигурируем git на работу по локальной сети

## Пользователь
- Нужно определиться с учётной записью — `<пользователем>`, под которым будет работать git.
- Узнать `<IP-адрес>` сервера, на котором будет храниться репозиторий.

## Сервер
- На сервере нужно зайти через терминал в домашнюю директорию пользователя, под которым будет работать git, и ввести в терминал `git init --bare <name>.git` — создать репозиторий `<name>`.
- После этого в домашней директории пользователя появится папка `<name>.git`, в которой будет храниться репозиторий.

## Клиент
- На стороне клиента нужно зайти в директорию, в которой будет храниться репозиторий, и ввести в терминал `git clone ssh://<пользователь>@<IP-адрес>/<path>/<name>.git` — склонировать удалённый репозиторий репозиторий, где `<path>` — полный путь к папке, в которой хранится репозиторий.
    - При наличии пароля для пользователя нужно ввести его после ввода команды.
- После этого в директории появится папка `<name>`, в которой будет храниться файлы репозитория.
